## **Introduction**

A public version of the Shelter Talk Api.
You can run this code locally or deploy it to a server.

## **Requirements**

- Go 1.23.2 or higher
- MongoDB 4.4 or higher

## **Installation**

```
git clone https://github.com/Thenemophilists/shelter-talk.git
cd shelter-api
go mod download
```
